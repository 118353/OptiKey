OptiKey

Free, Open Source, Assistive On-Screen Keyboard

Download this project as a .zip file Download this project as a tar.gz file

What is it?

OptiKey is a free assistive on-screen keyboard. It can be controlled by any combination of eye movement (using eye-tracking devices), mouse movement, keyboard presses, and mouse clicks giving everyone the ability to fully interact with a computer and communicate freely. It will run on computers, laptops and tablets running Windows.

Which eye trackers are supported?

Currently the following trackers are supported:

  • TheEyeTribe
  • GazeTracker Adding new input devices is straightforward. If you have an unsupported tracker get in touch and we can add support for it.

How do I get it?

Can I help with this project?

Absolutely. This is an open source project and I would love to build a community of volunteers around OptiKey. Email me: optikeyfeedback@gmail.com I have a list the length of my arm of ideas, features and improvements. Here's a sample;

  • Add support for more languages (dictionaries, as well as localising OptiKey)
  • Improve the algorithms which matches multi-key selections to dictionary words
  • Add word completion / incorrect spelling suggestions
  • Add next word suggestions
  • Support for accessible buttons, sip/puff tubes, brain potential detection, and lots of other human interfaces
  • and lots more...
