OptiKey

Free, Open Source, Assistive On-Screen Keyboard

What is OptiKey?

OptiKey is an free, on-screen keyboard to replace your physical keyboard. Instead of pecking away at keys you can interact with OptiKey by any combination of eye movement (using eye-tracking devices), mouse movement, keyboard presses, and mouse clicks giving everyone the ability to fully interact with a computer and communicate freely.

OptiKey can be used to simulate physical key presses (so you can type into your favourite email application, web browsers, etc), as well as speak your input out loud, or both.

How do I get it?

OptiKey is available for Windows: download the installer.

How do I use it?

A usage guide is coming very soon. In the mean time have a play and know that the settings (or "Management Console") are available by pressing ALT+M.

Troubleshooting

A troubleshooting page is coming very soon. In the mean time please email me with any problems you have and I'll help you directly: optikeyfeedback@gmail.com

If you're having trouble opening the Management Console using ALT+M then make sure OptiKey is the focussed window (click on it) and that OptiKey is not simulating a key press on any of the Shift/Alt/Win keys - they are treated as real key presses and will interfere with you pressing ALT+M.

Technical stuff

Which eye trackers are supported?

Currently the following trackers are supported:

  • TheEyeTribe
  • GazeTracker

Adding new input devices is straightforward. If you have an unsupported tracker get in touch and we can add support for it.

Which devices/operating systems are supported?

Anything which can run Windows Vista or later (including Windows 7, Windows 8 and the upcoming Windows 10).

Can I help with this project?

Absolutely. This is an open source project and I would love to build a community of volunteers around OptiKey. Email me: optikeyfeedback@gmail.com.
