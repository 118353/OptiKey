OptiKey

Free, open-source, assistive on-screen keyboard with speech capabilities

What is OptiKey?

OptiKey is an free (and open source) assistive on-screen keyboard for Windows, with built in speech capabilities.

It is designed to be a complete replacement for a physical keyboard (and mouse in the next release) and has been built to support interaction from assistive input devices (eye-tracking devices, sip/puff tubes, brain-computer interfaces, assistive buttons, etc), as well as standard input device (mouse, keyboard, microphone). The aim of OptiKey is to give everyone the ability to fully interact with a computer and communicate freely.

OptiKey can be used to simulate a physical keyboard (so you can type into your favourite applications), as well as speak out loud, or both.

How do I get it?

OptiKey is available for Windows: download the installer.

How do I use it?

A usage guide is coming very soon. In the mean time please install OptiKey and have a play with it. The settings (called the "Management Console") are available by pressing ALT+M.

Troubleshooting

A troubleshooting page is coming very soon. In the mean time please email me with any problems you have and I'll help you directly: optikeyfeedback@gmail.com

If you're having trouble opening the "Management Console" (using ALT+M), then make sure OptiKey is the focussed window (click on it) and that OptiKey is not simulating a key press on any of the Shift/Alt/Win keys - they are treated as real key presses and will interfere with you pressing ALT+M.

Technical stuff

Which eye trackers are supported?

Currently the following trackers are supported:

  • TheEyeTribe
  • GazeTracker

Adding new input devices is straightforward. If you have an unsupported tracker get in touch and we can add support for it.

Which devices/operating systems are supported?

OptiKey should work on any Windows pc/laptop/tablet running at least Windows Vista (the following should be fine; Vista, Windows 7, Windows 8, Windows 8.1 and the upcoming Windows 10).

Can I help with this project?

Absolutely. This is an open source project and I would love for there to be a community of volunteers around OptiKey. Email me: optikeyfeedback@gmail.com.
